## 一、本次修改内容

### 1. 新增 HTTP 拨号接口 `/dial`

**文件：** `main.js`

**位置：** HTTP 服务器处理逻辑

**原代码：** 无此接口

**新增代码：**
```javascript
// HTTP拨号接口 - 插件调用
if (url.pathname.includes('dial') && url.searchParams.has('number')) {
  const number = url.searchParams.get('number');

  // 验证手机号格式
  if (!/^1[3-9]\d{9}$/.test(number)) {
    res.writeHead(400, { 'Content-Type': 'application/json; charset=utf-8' });
    res.end(JSON.stringify({ success: false, error: '无效的手机号' }));
    return;
  }

  // 手机未连接时返回错误
  if (!phoneSocket || phoneSocket.readyState !== WebSocket.OPEN) {
    res.writeHead(200, { 'Content-Type': 'application/json; charset=utf-8' });
    res.end(JSON.stringify({ success: false, error: '手机未连接' }));
    return;
  }

  // 转发给手机端拨号
  phoneSocket.send(JSON.stringify({ type: 'dial', number }));

  // 将号码写入剪贴板（浏览器插件 Service Worker 无法写剪贴板）
  getClipboardy().then(lib => {
    lib.default.write(number).catch(() => {});
  });

  res.writeHead(200, { 'Content-Type': 'application/json; charset=utf-8' });
  res.end(JSON.stringify({ success: true, number: number }));
  console.log('[HTTP拨号] ' + number + ' (来自浏览器插件，已写入剪贴板)');
}
```

### 2. 全局 CORS 支持

**文件：** `main.js`

**位置：** HTTP 服务器顶部

**新增代码：**
```javascript
// 所有请求统一加 CORS 头（允许浏览器插件跨域访问）
res.setHeader('Access-Control-Allow-Origin', '*');
res.setHeader('Access-Control-Allow-Methods', 'GET, POST, OPTIONS');
res.setHeader('Access-Control-Allow-Headers', 'Content-Type');
if (req.method === 'OPTIONS') {
  res.writeHead(204);
  res.end();
  return;
}
```

### 3. `/dial` 接口 URL 匹配修复

**文件：** `main.js`

**原代码：**
```javascript
if (url.pathname === '/dial' && url.searchParams.has('number'))
```

**修改为：**
```javascript
if (url.pathname.includes('dial') && url.searchParams.has('number'))
```

**原因：** 浏览器扩展发出的请求，pathname 可能包含额外字符导致精确匹配失败。使用 `includes('dial')` 更健壮。

## 二、接口说明

### HTTP 拨号接口

**URL：** `GET http://127.0.0.1:35432/dial?number=13888888888`

**成功响应：**
```json
{ "success": true, "number": "13888888888" }
```

**失败响应（手机未连接）：**
```json
{ "success": false, "error": "手机未连接" }
```

**功能：**
1. 验证手机号格式
2. 通过 WebSocket 转发给已配对的 Android 手机
3. 将号码写入 Windows 剪贴板

---

## 三、完整调用流程

```
Edge 插件 content-script
    ↓ chrome.runtime.sendMessage({type:'dial', phone:'13888888888'})
background.js (Service Worker)
    ↓ fetch('http://127.0.0.1:35432/dial?number=13888888888')
PC 端 main.js (/dial 接口)
    ├→ 写入剪贴板 (clipboardy)
    └→ WebSocket 转发给手机端
         ↓
    Android 手机执行拨号
```

---

## 四、注意事项

1. **端口：** 35432（HTTP + WebSocket）
2. **仅限本机访问：** 监听 0.0.0.0，局域网内均可访问，请勿暴露公网
3. **剪贴板由 PC 端写入：** 浏览器插件 Service Worker 无法调用 clipboard API
4. **配对码固定：** 由 MAC 地址生成，重启不会改变
5. **CORS 已开放：** 所有 HTTP 响应均含 `Access-Control-Allow-Origin: *`
